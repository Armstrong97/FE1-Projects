Ordered List (ol)

Lista ordenada, lleva un paso a paso

  • type="1" Para lista por numeros, "A" alfabético y "I" # Romanos
  • start="20" La lista va a iniciar desde este número
  1. git status para revisar los cambios pendientes
  2. git add para añadir los archivos que han cambiado
  3. git commit -m para preparar la subida con msj
  4. git push origin master para subir sobre el branch original
  1. Git tutorial
  2. A Tag Info

Unordered List (ul)

Lista sin orden, va con bullet points

  • type="disc" para el circulo lleno
  • type="cicle" para el circulo vacío
  • type="square" para cuadrados rellenos
  • type="none" para quitar el ícono
